¿Cómo restaurar un mueble antiguo?

Restaurar un mueble antiguo es volver a la vida una pieza de madera con una historia por contar, sigue este artículo para conocer los pasos necesarios para devolver el encanto a tus muebles antiguos.

Materiales a utilizar:

  • Guantes de protección
  • Máscara para polvo
  • Barniz o tiente de tu elección
  • Lija gruesa
  • Paño de algodón
  • Sellador
  • Una brocha
  • Espátula
  • Pegamento para madera
  • Masilla para madera
  • Prensa

Paso 1:

Acondicionar el área de trabajo, escoge un lugar ventilado y espacioso donde puedas manipular y mover el mueble con facilidad.

Paso 2:

Antes de comenzar con la reparación es esencial que este se encuentre limpio, para esto, utilizaremos un paño de algodón un poco húmedo para remover el polvo. Si el mueble contiene grasa los mejor es utilizar un limpiador suave y dejarlo secar completamente

Paso 3:

Retirar el acabado del mueble, para esto utilizaremos primero la lija gruesa y nos ayudaremos de una espátula para los rincones más complicados. Finalmente, lijamos con la lija fina para eliminar residuos y lograr una superficie suave. Asegúrate de lijar en dirección de las fibras de la madera.

Paso 4:

Identificar los daños que tienen el muble, busca todo tipo de grietas, golpes o partes sueltas. Coloca masilla para madera en las grietas y hoyos (para esto utiliza la espátula) y deja secar. Finalmente, si se identifica alguna parte suelta, aplica pegamento para madera y deja secar fijando la pieza (para esto utiliza la prensa).

Paso 5:

Aplicar el sellador con una brocha cubriendo toda la superficie del mueble. Esto protegerá la madera y la prepara para colocar el tiente o barniz.

Paso 6:

Si prefieres un acabado natural, aplica una capa delgada de barniz con una brocha, lijando suavemente entre cada capa. Si lo que prefieres es que mueble tenga un tono personalizado, utiliza un tinte de color, aplicando capas delgadas con lijado entre sí.

Paso 7:

Si deseas un acabado más brillante o suave, puedes optar por utilizar algún producto que brinde el acabado de tu elección.

Paso 8:

Una vez que el mueble este completamente seco, monta cualquier parte que hayas desmontado y colócalo en el lugar que prefieras.

Ya puedes disfrutar de tu mueble rejuvenecido listo para muchos años más de uso.
